# Seguridad y reporte de errores

Si encuentras una falla de seguridad, error crítico de análisis, falso positivo grave o problema que pueda afectar la integridad del proyecto, repórtalo de forma responsable mediante un issue privado o contacto directo con el mantenedor del repositorio.

No publiques datos sensibles, información privada, credenciales, tokens, documentos reservados ni acusaciones personales sin soporte verificable.

Este proyecto debe usarse únicamente con fuentes públicas y bajo criterios de análisis responsable.
